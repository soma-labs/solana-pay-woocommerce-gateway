Solana Pay for your online WooCommerce store. Help customers pay instantly with a QR code or their Web 3.0 wallet, no fees.

== Installation ==
AUTOMATIC INSTALLATION

1. Log in to your WordPress dashboard, navigate to the Plugins menu and click \"Add New\". In the search field type \"solpay.store\" and click Search Plugins. After you\'ve found our plugin install it by clicking \"Install Now\".

2. Enable the plugin.

3. Go to WooCommerce -> Settings -> Payments and enable the \"Solana Payment\" payment method.

4. In the same admin panel click \"Manage\" to view and edit the plugin settings such as: Merchant Solana wallet, Enable development mode, etc.
